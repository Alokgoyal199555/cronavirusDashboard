## Build & Run
### Prerequisite
```bsh
$ npm install -g nodemon
$ npm install
```
### Development Mode

#### Run on terminal
```bsh
$ npm run watch
$ npm run dev
```

### Production Mode

#### Run on terminal
```bsh
$ npm run build
$ npm run production
```

#### Run as daemon using [PM2][pm2]
[pm2]: https://github.com/Unitech/pm2

```bsh
$ npm run build
$ npm start
```
